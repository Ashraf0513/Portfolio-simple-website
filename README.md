# Personal Portfolio Website

A simple personal portfolio website built using **HTML5** and **CSS3**. The website consists of two pages—**Home** and **Contact**—designed with a clean, responsive layout to showcase my profile and provide an easy way for visitors to get in touch.

## Features

* Clean and simple user interface
* Home page with personal introduction
* Contact page for reaching out
* Responsive design using basic CSS
* Well-structured HTML and CSS code

## Technologies Used

* HTML5
* CSS3

## Project Structure

```
portfolio/
├── index.html
├── contact.html
├── style.css
└── assets/
```

## Purpose

This project was created to practice the fundamentals of web development using HTML and CSS while building a basic personal portfolio website.
